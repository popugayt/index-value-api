# index-value-api
9-3-ПО-22
Семёнов Даниил, Рудаков Егор, Шашито Данил

Мобильное приложение предназначено для отслеживания актуальных курсов валют в режиме реального времени. Оно решает проблему быстрого и удобного доступа к достоверной информации о стоимости валют без необходимости посещать банковские сайты или искать данные в интернете. Пользователь может просматривать курсы основных мировых валют, сравнивать изменения за выбранный период и настраивать избранные валютные пары. Приложение создано для студентов, путешественников, предпринимателей и всех пользователей, которым важно оперативно получать информацию о валютных курсах для планирования расходов, покупок или финансовых решений.
